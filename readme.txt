Requirement

Use Python 3.6 or higher

Download xdelta3.exe and place it at 'C:\Versioning\xdelta3.exe' or change the location from the code

Download Python cst library and place it at 'C:\Program Files (x86)\CST Studio Suite 2022\AMD64\python_cst_libraries' or change the location from the code

create a file for logging at 'C:\Versioning\log.txt'

if you are not able to set the above files at given place then search for this path in the code and replace it with your own code.
